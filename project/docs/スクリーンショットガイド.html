📋 スクリーンショット撮影手順

  1. 各セクションのスクリーンショットエリアをクリック
  2. ブラウザの開発者ツールでデバイスサイズを調整
  3. デスクトップ・タブレット・モバイルの3つのサイズで撮影
  4. 画像ファイル名を「セクション名_デバイス名.png」の形式で保存
  5. 撮影した画像をこのガイドページに配置
